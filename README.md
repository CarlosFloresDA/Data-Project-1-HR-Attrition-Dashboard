🟦 README — HR Attrition Dashboard (Excel Project)
📌 Project Overview
This project analyzes employee attrition using an Excel-based interactive dashboard.
The goal is to identify who is resigning, why they are resigning, and which variables most influence attrition across different demographic, satisfaction, and job-related factors.
The dashboard was built using:
- PivotTables
- PivotCharts
- Slicers
- KPI cards
- A structured layout for clear storytelling
It includes analysis by Job Role, Department, Education Level, Age Band, OverTime, Distance From Home, Work-Life Balance, and Monthly Income.

🧠 Key Insights
- Young employees (25–34) show the highest attrition.
- Low income, poor work-life balance, and overtime strongly increase resignation rates.
- Sales and technical roles experience the highest turnover.
- Distance from home also correlates with higher attrition.

🏁 Conclusion
This project demonstrates practical HR analytics skills using Excel, including data preparation, KPI design, dashboard layout, and insight communication.
It serves as a portfolio-ready example of business-oriented data analysis.
